# Refract-CMS Project

## Prequisites

- Install NodeJS
- Install Docker

## Start Refract-CMS

```bash
npm install
npm run dev
```

## Development links

| Name               | Url                                            |
| ------------------ | ---------------------------------------------- |
| Dashboard          | <http://localhost:3000>                        |
| GraphiQL           | <http://localhost:3000/cms/graphql>            |
| GraphQL Playground | <http://localhost:3000/cms/graphql-playground> |
| Mongo Express      | <http://localhost:8081>                        |

## Run in production

```bash
npm run build
npm run start:prod
```
